Project Idea

Overview

Modern technologies give us a great opportunity to access information and experience different cultures. My project idea will combine these features together in a fun way. I suggest creating a smartphone application that would be contributing to gaining cultural experience as well as entertainment. The app will ask the user to choose a year (or maybe a decade or a century if we go further) and a country, and it will return a popular music album, a book and a movie from those times.

Motivation

Recently, streaming services like Netflix and Spotify became increasingly popular thanks to sophisticated search engines. These algorithms play a significant role in providing good user experience. Statistic shows that Netflix Recommendation Engine is so accurate that 80% of Netflix viewer activity is driven by personalised recommendation (Meltzer, R., n.d.). But what do we do when we want to explore fun things that are not based on our previous preferences? This application would be a helpful tool in exploring new cultures, history, new genres and artwork.

Description

The idea of this application is simple: the user will provide 2 inputs (a year and a country) and the app will return 3 outputs: a musical album, a movie and a book released in a chosen country and year. Once our parameters are chosen, the app will provide a brief description of the items. For example, short information about the musical artist, movie and a book plot. The application will show the images as well (album cover, movie poster and a book cover). An important thing would be linking these items to the servers where the users can access them. For example, it can be linked to Spotify, Netflix or YouTube movies and Booktopia respectively. Visually it will look like a clickable map of the world with the years list on the side. The app will have an interactivity element: users will be able to contribute and provide suggested artwork. In order to do that, the app will include a registration form and a section with the information they need to provide so a new item can be added to the database.

Tools and Technologies

The proposed application will combine front-end and back-end technologies. The back-end side will consist of a database with countries, years, movies, music albums and books. By providing 2 inputs the user will be able to retrieve the information from the database. As we want the application to provide multiple outputs, for each input we need to store multiple options, which makes the database even bigger. In order to store all the data we can use PostgreSQL. Cloudinary can be used to store images and list of our content. The front-end side of the app can be built on React and the back end can be built on Node (Barger R., 2021).

Skills Required

In order to complete this project, various skills must be developed. As this is a potential team project, good communication and soft skills are needed. The hard skills will include advanced programming using React, Node and JavaScript. It is also important to understand how all these parts will be connected and how the database will be organised. For the UX part of the project, such tools as Figma or Adobe XD will be used. I believe it will be feasible to find the skills and create the simplest version of this app within a short timeframe.

As I stated in my introduction, I adore art and design. I believe this project will help me to develop my skills both in front-end and back-end and give me a full understanding how to make a useful smartphone application. Once the project is completed, we will have a great entertaining and at the same time educational tool.
